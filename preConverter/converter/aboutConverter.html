ABOUT CONVERTER

If you are new to layout stuff, you will definitely regret not having tea beside you after reading this....!

The conversion is basically done from one keyboard layout to another.

Let's give some code to the conversions we make, Code [FT] => Two letter code which indicates from F layout to T layout.

Considering the top three most used layouts Qwerty, Dvorak, and Colemak. Q indicates Qwerty, D indicates Dvorak and C indicates Colemak. The different possible combinations are DQ, CQ, QD, CD, QC, DC.

Now lets talk about the conversion, QD implies Qwerty to Dvorak.

a,s,d,f keys in qwerty layout represent a,o,e,u letters in dvorak layout. The QD converter just converts the corresponding key in qwerty layout to a letter which represents the key in dvorak layout. So that "ishd" typed in qwerty layout represents "code" string in dvorak layout. And "code" typed in qwerty layout represents "jre." string in dvorak layout.

Pro tip: If you are a qwerty typer then you can take advantage of DQ, CQ. You will be typing in dvorak layout or colemak layout, thinking that it is qwerty and the display shows different text, that different text is the code, can be shared and converted by others in DQ or CQ converters.
